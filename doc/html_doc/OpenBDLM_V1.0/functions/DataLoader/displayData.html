displayData

PURPOSE

DISPLAYDATA Display on screen the content of the data in memory

SYNOPSIS

function displayData(data, misc)

DESCRIPTION 

DISPLAYDATA Display on screen the content of the data in memory

   SYNOPSIS:
     DISPLAYDATA(data)

   INPUT:
       data        - structure (required)
                         Two formats are accepted:

                           (1) data must contain two or three fields :

                               'timestamps' is a 1×N cell array
                               each cell is a M_ix1 real array

                               'labels' is a 1×N cell array
                               each cell is a character array

                               'values' is a 1×N cell array
                               each cell is a M_ix1 real array
                               This field is optional

                               N: number of time series
                               M_i: number of samples of time series i


                           (2) data must contain two or three fields:

                               'timestamps' is a M×1 array

                               'labels' is a 1×N cell array
                               each cell is a character array

                               'values' is a MxN  array
                               This field is optional

                               N: number of time series
                               M: number of samples

      misc         - structure (required)

   OUTPUT:
      Print message on screen

   DESCRIPTION:
      DISPLAYDATA displays on screen the content of the data in memory

   EXAMPLES:
      DISPLAYDATA(data)

   See also CHOOSETIMESERIES

CROSS-REFERENCE INFORMATION

This function calls:
  • convertMat2Cell CONVERTMAT2CELL Convert matrices in data.timestamps and data.values to cell array
This function is called by:
  • DataLoader DATALOADER Create a data file
  • chooseTimeSeries CHOOSETIMESERIES Request the user to select some time series
  • defineModel DEFINEMODEL Request user's input to define model
