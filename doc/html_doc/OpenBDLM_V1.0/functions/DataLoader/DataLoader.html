DataLoader

PURPOSE

DATALOADER Create a data file

SYNOPSIS

function [data, misc, dataFilename]=DataLoader(misc)

DESCRIPTION 

DATALOADER Create a data file

   SYNOPSIS:
     [data, misc, dataFilename]=DATALOADER(misc, varargin)

   INPUT:
      misc                 - structure
                             see the documentation for details about the
                             field in misc

      FilePath             - character (optional)
                               saving directory for data
                               default: '.'  (current folder)
   OUTPUT:
       data                - structure 
                               data must contain three fields :

                               'timestamps' is a M×1 array

                               'values' is a M×N array

                               'labels' is a 1×N cell array
                               each cell is a character array

                               N: number of time series
                               M: number of samples

     misc                  - structure
                             see the documentation for details about the
                             field in misc

     dataFilename          -  character

   DESCRIPTION:
      DATALOADER:
           - loads an existing processed database (DATA_*.mat file)
           - creates a new database (DATA_*.mat file) from .csv files

   EXAMPLES:
      [data, misc, dataFilename] = DATALOADER(misc)
      [data, misc, dataFilename] = DATALOADER(misc, 'FilePath', 'processed_data')

   EXTERNAL FUNCTIONS CALLED:
      displayDataBinary, loadData, editData

   See also DISPLAYDATABINARY, LOADDATA, EDITDATA

CROSS-REFERENCE INFORMATION

This function calls:
  • displayData DISPLAYDATA Display on screen the content of the data in memory
  • displayDataBinary DISPLAYDATABINARY Display the list of DATA_*.mat files
  • editData EDITDATA Control script to edit dataset (selection, resampling, etc..)
  • loadData LOADDATA Load data from csv file
  • saveDataBinary SAVEDATABINARY Save data in a binary Matlab .mat file
  • plotDataSummary PLOTDATASUMMARY Plot data amplitude, data time step, and data availability
This function is called by:
  • loadInteractive LOADINTERACTIVE Load a new project using interactive mode
