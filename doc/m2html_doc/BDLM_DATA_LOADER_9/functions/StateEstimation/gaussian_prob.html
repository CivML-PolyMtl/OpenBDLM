gaussian_prob

PURPOSE

GAUSSIAN_PROB Evaluate a multivariate Gaussian density.

 GAUSSIAN_PROB Evaluate a multivariate Gaussian density.
 p = gaussian_prob(X, m, C)
 p(i) = N(X(:,i), m, C) where C = covariance matrix and each COLUMN of x is a datavector
