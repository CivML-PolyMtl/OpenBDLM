BDLM_HeaderTemplate

PURPOSE

BDLM_HEADERTEMPLATE Create header template for BDLM Matlab functions

SYNOPSIS

function BDLM_HeaderTemplate(name)

DESCRIPTION 

BDLM_HEADERTEMPLATE Create header template for BDLM Matlab functions

   SYNOPSIS:
     BDLM_HEADERTEMPLATE(name)

   INPUT:
      name - name of the Matlab m-file (i.e name of the Matlab function)

   OUTPUT:
      Create a Matlab m-file named 'name'.m in current directory that
      contains the template

   DESCRIPTION:
      BDLM_HEADERTEMPLATE creates the header template for BDLM
      Matlab function

   EXAMPLES:
      BDLM_HEADERTEMPLATE('myfunction') will create a Matlab m-file named
    'myfunction.m' in current directory with the template

   EXTERNAL FUNCTIONS CALLED:
       N/A

   SUBFUNCTIONS:
       N/A
