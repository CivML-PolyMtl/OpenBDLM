This function calls:
  • DataLoader DATALOADER Create a data file

  • verificationDataStructure VERIFICATIONDATASTRUCTURE Verify formatting of data structure

  • modifyTrainingPeriod MODIFYTRAININGPERIOD Request user to modify training period

  • chooseProjectName CHOOSEPROJECTNAME Request user's input to choose a project name
  • convertCell2Mat CONVERTCELL2MAT Convert cells in data.timestamps and data.values to array
  • convertMat2Cell CONVERTMAT2CELL Convert matrices in data.timestamps and data.values to cell array
  • displayProjects DISPLAYPROJECTS displays on screen information about saved projects
  • initializeProject INITIALIZEPROJECT Initialize a project
  • printConfigurationFile PRINTCONFIGURATIONFILE Create and print a configuration file
  • printProjectDateCreation PRINTPROJECTDATECREATION Print date creation of the project
  • readConfigurationFile READCONFIGURATIONFILE Read configuration file
  • saveProject SAVEPROJECT Save full project (data, model, estimation, misc) in file
  • testFileExistence TESTFILEEXISTENCE Test existence of a file
  • computeInitialHiddenStates COMPUTEINITIALHIDDENSTATES Computes initial hidden states from smoother
  • state_estimation %%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
  • plotData PLOTDATA Plot amplitude and time step of time series data
  • plotEstimations PLOTESTIMATIONS Plot hidden states results
